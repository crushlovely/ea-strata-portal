Workflow Admin

Use this screen to toggle off/on certain modules and features of the Portal outside of the normal Case workflow.

The manual toggling done here is outside of the normal workflow controlled by the status of the Case. Please use this screen if you are sure you want to override the normal case workflow and manually turn on/off specific modules and features.

Time & Expense
  • Features
    • Time & Expense Entry
    • Common Benefit Acknowledgement
    • User Profile Survey (If Enabled)
Time & Expense - Show Jurisdiction Field
  • Features
    • Shows the (required) Jurisdiction dropdown field on Time & Expense Pop-ups.
    • Adds Jurisdiction field to both Time & Expense Reports.
    • Note: Requires Jurisdiction data table to be populated.
Time & Expense - Show Approved By Field
  • Features
    • Shows the (required) Approved By dropdown field on Time & Expense Pop-ups.
    • Adds Approved By field to both Time & Expense Reports.
    • Note: Requires Approved By data table to be populated.
Time Entry - Show Hourly Rate and Total Field
  • Features
    • Shows the Hourly Rate field on Time Entry Pop-up.
    • Shows the Total field on the Time Entry Pop-up.  This field automatically populates by multiplying the Hours * Hourly Rate.
    • Adds Hourly Rate and Total field to both Time & Expense Reports.
Time & Expense - Show Name of Case Memo Field
  • Features
    • Shows the Name of Case Memo dropdown field on Time & Expense Pop-ups.
    • Adds Name of Case Memo field to both Time & Expense Reports.
Expense Entry - Show Expense Type Field
  • Features
    • Shows the Expense Type dropdown field on Expense Pop-ups.
    • Adds the Expense Type field to Expense Reports.
    • Enforces import of the Expense Type field on Expense Entry Excel imports.
Time Entry - Show Activity Code Field
  • Features
    • Shows the Activity Code dropdown field on Time Entry Pop-ups.
    • Adds the Activity Code field to Time Entry Reports.
    • Enforces import of the Activity Code field on Time Entry Excel imports.
