Portal Enhancements: Bulk E-Signature Verification Requests and Reminders posted on 2/20/2026

  • You can find an updated User Guide on the Portal Documents tab.
  • To ensure you are seeing the latest version of the portal, you may need to perform a hard refresh.
    Windows: Press Ctrl + Shift + R OR Ctrl + F5 Mac: Press Cmd + Shift + R.

Bulk E-Signature Verification Requests

You can now send E-Signature Verification Requests to up to 500 plaintiffs at once using the Bulk Actions tab. Only plaintiffs with no outstanding items will be eligible for bulk signature. Please refer to Section V. Adding a Plaintiff to the Portal – G. Completing a Verification (page 23) in the User Guide

Reminders

  • Documents can be uploaded in bulk regardless of Plaintiff status.
  • SDWIS IDs can be updated via Import

Electronic Verification

The system will now default to sending the signature request to the email address listed on the Plaintiff Information tab. An alternative signer email may be provided if necessary.

Expert Disclosure

This question will no longer be required for submission to accommodate varying requirements.

Plaintiff Enrollment Report

Additional fields are now available on this report that were already stored on the Plaintiff Information tab such as plaintiff address, plaintiff email address, and original filing information.

Incomplete Items Report

A new report is now available that lists all incomplete items required for submission for each plaintiff. The report includes a full description of each incomplete item.
