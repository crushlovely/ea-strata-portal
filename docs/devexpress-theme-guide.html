DevExpress Theme Guide

The colors, typography, and control styling to reproduce the STRATA design system in a DevExpress ASP.NET Web Forms theme. Every value is copyable (click any hex or measurement) and mapped to the DevExpress control + state it applies to. Values are sourced directly from the Figma variables in Settlement-Redesign.

How to use this document

This guide pairs a visual spec (live-rendered swatches and buttons that match the design exactly) with copyable values and a DevExpress mapping so the theme can be built directly against ASPx controls.

Click any hex code or measurement to copy it to your clipboard.
DevExpress target: the portal uses classic ASP.NET Web Forms controls (ASPxButton, ASPxGridView, ASPxTextBox, ASPxPopupControl, …). In the DevExpress Theme Builder these are styled per control; button state classes are .dxbButton (default), .dxbButtonHover, .dxbButtonPressed (active), .dxbButtonFocused, and .dxbDisabled.

Buttons

Six intents — Primary, Secondary, Outline, Danger, Simple (text/link), and Icon — each in three sizes (Large, Medium, Small) and five states (Default, Hover, Focus, Active, Disabled). Hover/Focus/Active share the same fill on most intents; where a subtle vertical gradient is used the spec lists both the top color (bg) and bottom color (bg₂).

State → DevExpress class: Default → .dxbButton · Hover → .dxbButtonHover · Focus → .dxbButtonFocused · Active/pressed → .dxbButtonPressed · Disabled → .dxbDisabled. Map Primary to the default/accent button, Danger to destructive actions, Outline & Secondary to standard buttons, and Simple to link-style buttons.
